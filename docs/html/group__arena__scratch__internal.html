#include <arena_scratch.c>

Initialize a single scratch slot with its own arena.

This internal helper sets up a t_scratch_slot by initializing the internal arena with the given slot_size. The arena is marked as growable, and the slot is initially marked as not in use.

On failure, an error is reported using arena_report_error(), indicating the index of the slot that failed to initialize.

Parameters
slotPointer to the scratch slot to initialize.
slot_sizeSize of the arena to allocate for the slot.
indexIndex of the slot (used for debugging/logging).
Returns
true if the slot was successfully initialized, false otherwise.
Note
This function is called during scratch_pool_init() to initialize each slot in the scratch pool.
