Detailed Description

Arena state inspection and control utilities.

Author
Toonsa
Date
2025

This module provides utility functions for inspecting and manipulating the state of a memory arena without modifying its underlying buffer directly. These functions support read-only queries (e.g., used, remaining, peak usage), as well as efficient bulk memory rollback via markers (arena_mark() and arena_pop()) and full arena reset.

Features:

  • Query memory usage (arena_used, arena_remaining, arena_peak).
  • Save and restore allocation state using arena_mark() and arena_pop().
  • Fully reset the arena for reuse using arena_reset().

These utilities are thread-safe and lock the arena internally before accessing or modifying its state.

Note
These functions are designed for convenience and performance in scenarios like:
  • Frame-based memory reuse (e.g., in games or simulations).
  • Temporary scopes for allocation and rollback.
  • Profiling and diagnostics (e.g., monitoring peak usage).
