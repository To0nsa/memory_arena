Assign a human-readable label to an arena for debugging and introspection.

Assign a human-readable label to an arena for debugging and introspection.This function sets the debug.label field of the specified arena to the given string. The label can later be used for:

  • Logging and error messages
  • Visualizers or diagnostics tools
  • Identifying specific arenas during profiling or debug output

The function does not copy or allocate memory for the label; it simply stores the pointer. Make sure the label string remains valid for the lifetime of the arena.

Parameters
arenaPointer to the arena to label.
labelNull-terminated string to assign as the arena’s label.
Returns
void
Note
If arena is NULL, the function does nothing. If label is NULL, any existing label is cleared.
