Detailed Description

Internal utilities and helpers for arena memory management.

Author
Toonsa
Date
2025

This header defines functions and macros used internally by the arena system. It includes synchronization macros (ARENA_LOCK, ARENA_UNLOCK), metadata utilities, and default logic for peak tracking and growth behavior.

These APIs are not intended for external use and may change without notice.
